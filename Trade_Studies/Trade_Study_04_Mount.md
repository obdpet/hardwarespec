# Carpybara — Trade Study #4: 거치 방식 선정

> ⚠ **현재 유효 결론은 Rev C (문서 하단)** — **Beanbag + Magnet (MagSafe-style)** 채택.
> Rev A(본문 1~11절)는 VHB 접착을, Rev B는 Dual Lock을 권고했으나, 모두 대시보드에 영구 접착제를 남긴다는 문제가 확인되어 2026-04-16 Rev C에서 최종 재평가됨. Rev A/B는 이력 보존용.

> **결정 사항**: Carpybara를 차량 내에 어떤 방식으로 거치할 것인가?  
> **작성일**: 2026-04-14 (Rev A) / 2026-04-15 (Rev B 재평가)  
> **배경**: Trade Study #2에서 송풍구 클립(≤2.8") 기준으로 설계했으나, 거치 방식 변경이 결정됨. 대시보드 또는 앞유리 부착 중 구체적 방식을 확정해야 함.

---

## 0. Trade Study란?

하드웨어 엔지니어가 부품이나 설계 방향을 결정할 때 사용하는 **표준 의사결정 방법론**입니다.

핵심 원리:
1. **감으로 고르지 않는다** — 평가 기준을 먼저 정하고, 점수로 비교한다
2. **가중치가 결론을 만든다** — 같은 부품이라도 프로젝트 상황에 따라 정답이 달라진다
3. **문서로 남긴다** — "왜 이걸 골랐는가"를 나중에 설명할 수 있어야 한다

> Trade Study는 "정답을 찾는 도구"가 아니라 **"오답을 걸러내는 도구"**입니다.
> 최종 판단은 여전히 사람이 하지만, 근거가 투명하게 남습니다.

## 1. 문제 정의 (Problem Statement)

Carpybara는 차량 대시보드에 설치하는 캐릭터 가젯이다.  
기존 설계는 송풍구 클립 거치를 전제로 했으나, 다음 이유로 변경을 검토한다:

- 송풍구 클립은 차량 모델마다 송풍구 형상이 달라 범용성이 낮음
- 디스플레이 크기 제약(≤2.8")이 제품 경험을 제한
- 에어컨 바람을 차단하여 사용자 불만 가능성

**결정해야 할 것**: 대시보드 접착, 앞유리 흡착, 마그네틱 등 구체적 거치 방식 확정

## 3. 후보 거치 방식 (Candidates)

---

### Option A: 대시보드 VHB 접착 (스냅인 크래들)

```
[구조]
대시보드 표면 ← 3M VHB GPH-160 테이프 ← 마운트 베이스(영구 부착)
                                         ↑
                                    Carpybara 본체 (스냅인/스냅아웃)
```

- **접착제**: 3M VHB GPH-160 (내열 149°C, 아크릴 폼)
- **참고 제품**: Garmin/Viofo/Rexing 대시캠 마운트 (업계 표준 방식)
- **장점**: 강력한 접착, 진동 없음, 법적 제한 없음, 대시캠 업계 검증
- **단점**: 제거 시 열풍기 필요, 잔여물 발생. 일부 곡면/소프트터치 대시보드 호환 불가
- **BOM 추가**: ~$0.05 (VHB 테이프)

---

### Option B: 앞유리 흡착컵

```
[구조]
앞유리 유리면 ← 실리콘 흡착컵 ← 조절 암(arm) ← Carpybara 본체
```

- **참고 제품**: TomTom GPS, 구형 Garmin GPS, 일부 대시캠
- **장점**: 탈착 자유로움, 잔여물 없음, 모든 차량 호환
- **단점**: 고온에서 흡착 실패, 암(arm) 진동, **미국 다수 주에서 불법** (아래 상세)
- **BOM 추가**: ~$1~2 (흡착컵 + 암)

---

### Option C: 대시보드 VHB + 마그네틱 스냅

```
[구조]
대시보드 표면 ← 3M VHB ← 마운트 베이스(자석 내장, 영구 부착)
                              ↕ 자력으로 탈착
                         Carpybara 본체(스틸 플레이트 또는 자석 내장)
```

- **참고 제품**: Nextbase Click&Go PRO 대시캠, MobArmor 차량 마운트
- **장점**: 원핸드 탈착, VHB 안정성 + 마그네틱 편의성. "매일 떼었다 붙이는" 사용 패턴에 최적
- **단점**: 인클로저에 페라이트 인서트 또는 네오디뮴 자석 내장 필요. BOM 증가. PCB 근접 자기장 검증 필요
- **BOM 추가**: ~$1~3 (네오디뮴 자석 + 스틸 플레이트 + VHB)

---

### Option D: 대시보드 클립/브래킷

```
[구조]
대시보드 가장자리 ← 스프링 클립 ← Carpybara 본체
```

- **참고 제품**: ProClip USA (차량 특화 브래킷)
- **장점**: 접착제 불필요, 견고한 고정
- **단점**: 차량 모델별 대시보드 형상에 따라 설치 불가. 최신 차량은 대시보드에 클립 포인트 없음. 범용 제품에 부적합
- **BOM 추가**: ~$0.50~1 (플라스틱 클립)

## 3.5. 핵심 발견: 앞유리 거치의 미국 법적 리스크

조사 중 **제품 사업에 치명적인 법적 리스크**가 확인되었습니다.

### 캘리포니아 (Irvine 본사 소재지)

**California Vehicle Code § 26708**에 의거:

| 기기 유형 | 허용 위치 |
|----------|---------|
| GPS/네비게이션 | 앞유리 **운전석 쪽 하단 모서리 5인치 정사각형** 또는 **조수석 쪽 하단 모서리 7인치 정사각형** |
| 블랙박스(dashcam) | 위와 동일 + **상단 중앙 5인치 정사각형** |
| 일반 디스플레이 가젯 | **명시적 허용 조항 없음** — 하단 모서리만 조건부 가능 |

→ Carpybara를 앞유리에 거치하면 **하단 모서리에만 설치 가능** — 운전자 시선에서 벗어나 제품 가치 훼손.

### 미국 전체

| 구분 | 주 수 | 내용 |
|------|:---:|------|
| **앞유리 부착물 전면 금지** | **28개+** | AL, AR, CT, DE, GA, ID, IA, KS, KY, LA, MT, NE, NJ, NM, ND, OK, OR, PA, RI, SC, SD, TX, VA, WA, WV, WI, WY, DC 등 |
| 위치 제한 | ~10개 | NY (좌하단만), FL (운전자 손 닿지 않는 곳), MN (하단만) 등 |
| 비교적 자유 | ~12개 | AZ, HI, IN, MD, NV, UT 등 |

**결론: 앞유리 흡착컵을 기본 거치 방식으로 채택하면, 미국 50개 주 중 ~28개 주에서 사용 자체가 불법.** Amazon US 판매 제품으로서 치명적인 제품 리스크.

> **대시보드 거치는 어떤 주에서도 규제하지 않습니다.** CVC § 26708은 "앞유리(windshield)"에 부착하는 물체만 규제합니다.

## 7. 민감도 분석 (Sensitivity Analysis)

### 시나리오 1: "법적 합법성이 덜 중요하다면?" (C1 가중치 5→2)

미국 외 시장(법적 제한 없는 국가)을 1차 타겟으로 하는 경우.

| 후보 | 기존 총점 | 차감분 | 변경 후 총점 |
|------|:---:|:---:|:---:|
| A: VHB 접착 | 117 | −5×3 = −15 | **102** |
| B: 흡착컵 | 77 | −1×3 = −3 | **74** |
| C: VHB+마그네틱 | 116 | −5×3 = −15 | **101** |
| D: 클립 | 96 | −5×3 = −15 | **81** |

→ **법적 합법성을 낮춰도 B는 여전히 최하위.** 안정성·내구성 문제가 법과 별개로 B를 끌어내림.

### 시나리오 2: "탈착 편의성이 핵심이라면?" (C7 가중치 2→6)

사용자가 매일 기기를 차에서 떼어가는 사용 패턴인 경우.

| 후보 | 기존 총점 | 추가분 | 변경 후 총점 |
|------|:---:|:---:|:---:|
| A: VHB 접착 | 117 | +2×4 = +8 | **125** |
| B: 흡착컵 | 77 | +5×4 = +20 | **97** |
| C: VHB+마그네틱 | 116 | +5×4 = +20 | **136** |
| D: 클립 | 96 | +3×4 = +12 | **108** |

→ **C(마그네틱)가 136으로 역전 1위!** 탈착이 중요하면 마그네틱 방식이 최적. A는 "영구 설치" 특성 때문에 뒤처짐.

### 시나리오 3: "인클로저를 최대한 단순하게" (C8 가중치 2→5)

케이스 설계 복잡도를 최소화하고 싶은 경우.

| 후보 | 기존 총점 | 추가분 | 변경 후 총점 |
|------|:---:|:---:|:---:|
| A: VHB 접착 | 117 | +5×3 = +15 | **132** |
| B: 흡착컵 | 77 | +3×3 = +9 | **86** |
| C: VHB+마그네틱 | 116 | +3×3 = +9 | **125** |
| D: 클립 | 96 | +4×3 = +12 | **108** |

→ **A의 리드 확대 (132 vs 125).** 단순 설계 우선 시 VHB 접착 확정적.

### 민감도 분석 결론

- **A(VHB)와 C(마그네틱)는 사실상 동급** — 1점 차이는 오차 범위
- 탈착이 중요하면 C가 역전, 단순 설계가 중요하면 A가 벌어짐
- **B(흡착컵)는 어떤 시나리오에서도 최하위** — 법적 리스크 + 내구성 문제의 이중 약점

### B(앞유리 흡착)가 부적합한 이유

> **앞유리 거치는 기본 거치 방식으로 채택 불가.** 옵션 액세서리로도 법적 리스크 면에서 권장하지 않음.

- 2018년 이후 차량 대시보드는 연속 곡면 디자인이 대세 — 클립 포인트가 없는 차량 증가

# Rev C — 2026-04-16 최종 재평가

> **이 섹션이 현재 유효 결론입니다.** Rev A/B 본문은 의사결정 이력 보존용으로 유지되며, Option G (Beanbag + Magnet, MagSafe-style)가 최종 채택안입니다.

## C-1. 재평가 트리거 (Why Revisit Again)

Rev B 결론(Dual Lock SJ3550) 채택 후, 추가 사용자 피드백으로 제기된 근본적 문제:

> *"VHB 기반이란 건 여전히 스티커로 되어있다는 얘기지?"*  
> *"나는 차량 대시보드에 뭔가 남는게 싫거든. 차량은 비싼 제품이라 다른 사용자들도 그걸 원할 것 같아."*

**핵심 insight**: Dual Lock은 "기기 탈착 UX"는 해결했으나, **베이스 측 VHB 스티커(25×25mm)가 대시보드에 영구 부착**되는 문제는 그대로. 차량은 소비자의 고액 자산이며, "대시보드 원상 보존"은 제품 UX의 필수 요건임이 재확인됨.

## C-3. 신규 후보: Option G (Beanbag + Magnet, MagSafe-style)

본 재평가의 핵심 신규 후보. Rev B의 "Option C (VHB+마그네틱)"이 탈락했던 이유(VHB 베이스)를 **베이스를 Beanbag으로 대체**하여 해결.

### 구조

```
[대시보드] ← 무접촉 (Beanbag은 얹기만)
[Beanbag 베이스: PU 외피 + 비드 충전, 바닥 고무 논슬립 (100×80×20mm, ~150g)]
[Beanbag 상면에 매립된 자석 어레이 (MagSafe 호환 ring 또는 N42SH disc ×4)]

[Carpybara 케이스 바닥: 강판(SUS430) 또는 매칭 자석 인서트]
```

### 핵심 스펙

| 항목 | 스펙 |
|------|------|
| 자석 등급 | **N42SH** (150°C) 또는 **N45SH** (Apple MagSafe 표준, NiCuNi 도금) |
| 자력 | 800~1100 gf (MagSafe ring 규격 동일, Carpybara 150g × 2G 마진 여유) |
| 비드백 외피 | PU/PVC 블랙 (UV 저항, 내열 90°C) |
| 비드백 충전 | 재활용 플라스틱 비즈 + 약간의 모래 (진동 흡수) |
| 자석-PCB 이격 | ≥15mm (WiFi/BT 안테나 간섭 방지) |
| 추가 BOM | 자석 ~$1.5~2.5 + 비드백 ~$2~3 = **+$3.5~5.5** |

### 사용자 UX 시나리오

1. 박스 개봉 → Carpybara + Beanbag 베이스 + USB-C 케이블 + 2장 매뉴얼

3. Carpybara를 Beanbag 상면에 톡 — 자석 원터치 결합
4. 차량 변경 시 Beanbag만 집어들어 다른 차량으로 이동 — **대시 완전 원상**

- **C2 주행 안정성**: Magnet이 Gel보다 우위 (Gel은 전단력에서 슬립, Magnet은 수직 분리만 허용)
- **C3 내열**: Gel은 고온 녹음 리스크, N42SH 자석은 150°C 안정
- **C7 탈착**: Magnet 원터치(5점), Gel은 떼어야 함(4점)
- **C9 BOM**: Magnet 조합이 가장 비쌈 (+$3.5~5.5) — 그러나 가중치 1로 영향 제한

## C-8. 최종 권고 (Rev C)

### 1순위: Option G — Beanbag + Magnet (MagSafe-style)

**v1 거치 방식을 이것으로 확정.**

핵심 근거:
1. 가중치 적용 후 **118/140 (84.3%)** 1위, 2위 대비 10점 차
2. **R0 (대시 원상보존) 완전 통과** — 사용자 핵심 인사이트 직접 해결
3. 주행 안정성 양호 (자력 + 웨이트 이중 결합)
4. 내열 여유 (N42SH 150°C, SoCal 80°C+ 충분 마진)
5. 탈착 UX 최고 — MagSafe급 원터치
6. 민감도 분석 전 시나리오에서 1위 유지 (robust)
7. **Amazon 리스팅 차별화 포인트** — "No Damage Dashboard Mount. MagSafe-Compatible Base."

### 2순위 (백업): Option C' (Beanbag + Gel)

다음 조건에서 C'로 fallback:
- 자석 부품 공급 중단
- FCC SDoC EMC 테스트 실패
- 원가 압박 극단 (BOM -$4 필요)

### 비권고: Option E (Vent Clip)

- 송풍 직격으로 디스플레이 온도 스트레스 반복
- 차종별 vent 형상 편차 큼
- Beanbag 기반 옵션이 모든 측면에서 우위
