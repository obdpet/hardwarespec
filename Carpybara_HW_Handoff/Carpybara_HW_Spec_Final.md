### 1-6. 케이스 소재 및 거치 방식

| 항목 | 스펙 |
|------|------|
| **소재** | **PC-ABS 블렌드** (ABS 단독 사용 불가) |
| **이유** | 대시보드 직사광선 시 80°C+ 도달, ABS 단독은 연속 사용 온도 ~80°C로 경계선 |
| **참고** | PC-ABS: ~100°C / Pure PC: ~120°C |
| **거치 방식** | **Beanbag + Magnet (MagSafe-style)** — 대시 무접촉 Beanbag 베이스 + 원터치 자석 결합 |
| **자석 등급** | **N45SH (MagSafe 규격)** 또는 최소 N42SH (150°C 안정) — Pull force 800~1100 gf |
| **필수 요건 (R0)** | **대시보드에 영구 접착제/잔여물 금지** — 차량 원상 보존 |
| **추가 규제** | California Prop 65 대응 — 소재 RoHS 적합성 서류 확보 필요 |

> **변경 이력**:
> - Trade Study #4 Rev A: 송풍구 클립 → 대시보드 VHB 접착
> - Rev B: VHB → 3M Dual Lock SJ3550 (탈착 UX 개선)
> - **Rev C (현행)**: Dual Lock → **Beanbag + Magnet**. Dual Lock도 베이스에 VHB 잔여물을 남기는 문제 해결. 대시 완전 무접촉. 자석은 원터치 탈착 UX 제공.
> - 앞유리 흡착컵은 미국 28개+ 주에서 불법이므로 채택 불가 (Rev A 결정 유지)

### STEP 3. PCB 설계 고려사항 (Design for Manufacturability)

#### 3-1. 레이어 수

| 레이어 | 비용 | 적합 상황 |
|--------|------|----------|
| 2레이어 | 기준 | 단순한 회로, **Carpybara 수준에 적합** |
| 4레이어 | 약 2배 | 고속 신호, EMC 중요 설계 |

#### 3-2. SMD vs Through-hole

- **SMD(표면실장)**: 자동화 라인 가능, 소형화, **양산 표준**
- **Through-hole**: 수작업 필요, 비용 증가 → 양산에서는 커넥터류 외 최소화

#### 3-3. 테스트 포인트 (Test Point)

- 양산 시 각 기판의 전기적 검사를 위한 **테스트 패드**를 PCB에 반드시 설계
- 없으면 불량 기판 선별이 불가능하거나 수작업으로 진행해 비용 폭증

#### 3-4. 반드시 PCB에 포함할 회로 체크리스트

- [ ] 전원 보호: 역전압 방지 다이오드 (USB 잘못 연결 대비)
- [ ] ESD 보호: USB-C, 외부 커넥터 핀 모두
- [ ] 디커플링 캐패시터: MCU 전원 핀 주변 (100nF × 여러 개)
- [ ] 리셋 버튼 (개발 및 A/S용)
- [ ] 상태 표시 LED (전원, 통신 등)
- [ ] 안테나 클리어런스 영역

### STEP 4. 인증 (Certification) 계획

인증은 **설계 단계에서부터** 고려해야 합니다. 나중에 추가하면 회로 수정이 필요할 수 있습니다.

**우선순위: FCC → CE → KC (미국 1차 시장 기준)**

| 인증 | 국가 | 우선도 | 내용 | 비용 추정 |
|------|------|--------|------|---------|
| **FCC** | 미국 | **최우선** | 전파 방출 기준, Amazon 판매 필수 | $2,000~5,000 |
| CE | 유럽 | 2순위 | 안전/전자파, 유럽 판매 필수 | $2,000~5,000 |
| KC | 한국 | 3순위 | 전파/안전 인증 | $1,000~3,000 |
| RoHS | 전세계 | 설계 시 반영 | 유해물질 규제 | 부품 인증서로 대체 |
| **California Prop 65** | 캘리포니아 | **미국 내 필수** | 유해물질 경고 라벨 | 소재 확인 후 라벨 부착 |

**FCC 비용 절감 전략**
- ESP32-WROOM-32E는 이미 FCC ID 인증 완료 모듈 → 전체 제품은 **FCC SDoC(Supplier's Declaration of Conformity)** 방식으로 진행 가능
- SDoC는 FCC ID보다 훨씬 저렴하고 빠름 (인증 기관 제출 불필요, 자체 선언)
- 단, 제품이 FCC 규정을 실제로 만족해야 하므로 사전 EMC 테스트는 필수

**Amazon 판매 시 추가 요구사항**
- FCC/IC 인증 또는 SDoC 서류 제출 필수
- 배터리 포함 제품은 UN38.3 인증 필요 (Carpybara는 해당 없음 — USB 전원)
- Product liability insurance (제품 배상책임보험) 권장 ($1M 이상)

### STEP 5. 신뢰성 검증 (Reliability Testing)

양산 전 반드시 거쳐야 할 테스트:

| 테스트 | 목적 | 방법 |
|--------|------|------|
| **열 사이클 테스트** | 온도 변화에 의한 부품/납땜 크랙 | -20°C ↔ 85°C 반복 100회 이상 |
| **진동 테스트** | 차량 주행 진동 내구성 | IEC 60068-2-6 규격 참조 |
| **고온 장시간 테스트** | 85°C 환경에서 48시간 이상 연속 동작 | 오븐 or 챔버 |
| **EMC 사전 테스트** | FCC SDoC 전 자체 검증 | 인증 기관 사전 컨설팅 또는 Pre-scan 서비스 |

**Southern California 근처 EMC 테스트 기관 (참고)**
- Elite Electronic Engineering (Downers Grove) — 원격 예약 가능
- Intertek / UL — LA/OC 지역 사무소 보유
- **저비용 대안**: 중국 제조사에 EMC pre-scan 요청 (JLCPCB 연계 테스트 서비스)

### STEP 6. 초도 생산 (Pilot Run) 프로세스

| 단계 | 수량 | 목적 |
|------|------|------|
| **EVT** (Engineering Validation Test) | 5~10개 | 회로 동작 검증, 버그 수정 |
| **DVT** (Design Validation Test) | 20~50개 | 신뢰성/환경 테스트, 케이스 피팅 |
| **PVT** (Production Validation Test) | 100~200개 | 양산 라인 검증, 불량률 측정 |
| **MP** (Mass Production) | 목표 수량 | 본 양산 |

> 처음 하는 경우, EVT→DVT 사이에서 회로 수정이 1~2회 발생하는 것이 일반적입니다.  
> 이 비용과 기간을 프로젝트 일정에 반드시 포함하세요.

## PART 3. LLM 기술 경쟁으로 스펙 확정하는 법

### 각 LLM의 특성과 역할 분담

| LLM | 강점 | Carpybara 스펙 수립에서의 역할 |
|-----|------|------------------------------|
| **Perplexity** | 실시간 웹 검색, 최신 부품 가격·재고 조회 | 부품 단가 조사, 대안 부품 탐색, 경쟁 제품 스펙 리서치 |
| **Gemini** | Google 검색 연동, 긴 컨텍스트 처리 | 데이터시트 분석, 다수 부품 비교, 기술 표준 조사 |
| **Claude** | 논리적 추론, 장문 구조화, 일관성 검토 | 전체 스펙 정합성 검토, 트레이드오프 분석, 최종 문서 작성 |

### 권장 워크플로우

```
1. Perplexity로 부품 리서치
   └─ "ESP32-WROOM-32E 16MB 현재 Digi-Key 단가와 재고는?"
   └─ "2.5인치 IPS 85도 내열 패널 미국 내 공급업체는?"
   └─ "현재 중국산 전자부품 미국 관세율은?"

2. Gemini로 기술 검증
   └─ 데이터시트 업로드 후 "이 부품이 차량 환경에 적합한가?"
   └─ 여러 대안 부품의 스펙 표 비교 요청

3. Claude로 최종 정합성 검토 및 문서화
   └─ 전체 스펙을 붙여넣고 "모순되거나 누락된 항목이 있는가?"
   └─ 최종 스펙 문서 작성 및 리뷰
```

### LLM 기술 경쟁 진행 방식

각 LLM에게 **동일한 질문**을 던지고 답변을 비교합니다.

**질문 예시 템플릿**:
```
[프로젝트 컨텍스트]
미국 캘리포니아 Irvine 기반 스타트업.
차량 대시보드용 IoT 가젯, 소매가 $50 이하, 1차 시장 미국(Amazon), 양산 목표 10,000개.
현재 디스플레이 후보: 2.5인치 IPS TFT (드라이버 IC 미확정).
중국 제조 + 미국 수입 예정.

[질문]
1. 이 디스플레이가 차량 대시보드 환경(-20~85°C)에 적합한가?
2. 대안 부품이 있다면 무엇인가? 비용과 성능 비교.
3. PCB 설계 시 이 디스플레이와 관련하여 놓치기 쉬운 함정은?
4. 미국 FCC SDoC 진행 시 이 디스플레이 관련 주의사항이 있는가?
```

**합의점 도출 방법**:
- 세 LLM이 **모두 동의하는 항목** → 확정
- **의견이 갈리는 항목** → 해당 데이터시트 직접 확인 또는 전문가(EE 엔지니어) 자문
- LLM은 틀릴 수 있으므로, 핵심 수치(온도 범위, 전류 소비 등)는 항상 **공식 데이터시트**로 최종 확인

## 체크리스트 — 스펙 확정 전 최종 점검

**기술 스펙**
- [ ] 모든 부품의 동작 온도가 -20°C ~ 85°C 커버하는가?
- [ ] 핵심 부품마다 2nd Source(대체 공급처)가 있는가?
- [ ] PCB에 테스트 포인트가 설계되어 있는가?
- [ ] USB-C 포트에 ESD 보호 회로가 있는가?

**비용 및 공급망**
- [ ] 관세 포함 BOM 총합이 목표 소매가의 40% 이내인가?
- [ ] 중국산 관세 시나리오별 수익성 계산이 완료되었는가?
- [ ] 금형 초기 비용이 재무 계획에 포함되어 있는가?

**인증 및 규제 (미국 기준)**
- [ ] FCC SDoC 진행 계획 및 EMC pre-scan 일정이 있는가?
- [ ] California Prop 65 소재 적합성 확인이 완료되었는가?
- [ ] Amazon 판매 시 필요한 인증 서류가 준비되었는가?

**생산 프로세스**
- [ ] EVT → DVT → PVT 일정이 로드맵에 반영되어 있는가?
- [ ] 제조 파트너(JLCPCB 또는 대안)가 선정되었는가?
- [ ] 관세 리스크에 따른 제조 국가 전환 기준이 정해져 있는가?
